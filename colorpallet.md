# LuckyGeneMDx Color Palette

Based on the CSS files provided (`css/portal.css` and `css/admin.css`), both panels use the **Microsoft Fluent Design System** color palette. They share a core foundation but the Admin panel adds a distinct red accent.

### **User Portal Palette** (`css/portal.css`)
Designed for a clean, patient-friendly experience using Microsoft Blue as the primary action color.

| Color Role | Light Mode (Default) | Dark Mode |
| :--- | :--- | :--- |
| **Primary Brand** | `#0078D4` (Microsoft Blue) | `#479ef5` (Light Blue) |
| **Primary Dark** | `#005A9E` | `#0078d4` |
| **Background** | `#f3f2f1` (Light Gray) | `#1b1b1b` (Dark Gray) |
| **Card Surface** | `#ffffff` (White) | `#252525` (Acrylic Dark) |
| **Text Primary** | `#201f1e` (Near Black) | `#ffffff` (White) |
| **Text Secondary** | `#605e5c` (Gray) | `#d0d0d0` (Light Gray) |
| **Success** | `#107C10` (Green) | *Inherited* |
| **Warning** | `#d83b01` (Orange-Red) | *Inherited* |
| **Error** | `#a80000` (Dark Red) | *Inherited* |

### **Admin Panel Palette** (`css/admin.css`)
Identical to the User Portal but introduces a **Red Accent** to visually distinguish the administrative environment (used in the top border and badges).

| Color Role | Light Mode (Default) | Dark Mode |
| :--- | :--- | :--- |
| **Primary Brand** | `#0078D4` (Microsoft Blue) | `#479ef5` (Light Blue) |
| **Admin Accent** | **`#d13438` (Microsoft Red)** | **`#e05d5d` (Light Red)** |
| **Background** | `#f3f2f1` | `#1b1b1b` |
| **Card Surface** | `#ffffff` | `#252525` |
| **Text Primary** | `#201f1e` | `#ffffff` |
